A quoi sert Javascript?

JavaScript permet d'ajouter de l'interactivité aux pages web, rendant l'expérience utilisateur plus dynamique et réactive.

Qu'est-ce qu'ECMAScript?

ECMAScript est un ensemble de normes concernant les langages de programmation de type script, standardisées par Ecma International dans le cadre de la spécification ECMA-262. Il s'agit d'un standard implémenté dans des langages comme JavaScript ou ActionScript.

Qu'est-ce que le DOM?

Le Document Object Model (DOM) est une interface de programmation permettant de structurer des documents HTML et XML. Il offre aux programmeurs un accès aux composants d'un projet Web pour faciliter l'édition et la manipulation du contenu, des attributs et des styles.

Qu'est-ce qu'une fonction en javascript?

Réponse à la question 4: Les fonctions sont des procédures JavaScript, un ensemble d'instructions effectuant une tâche ou calculant une valeur. Il est nécessaire de les définir avant de les utiliser dans la portée souhaitée.

Qu'est-ce qu'un évènement en javascript?

Les événements sont des actions ou occurrences dans le système que vous programmez. Ils vous informent afin que vous puissiez y répondre. Par exemple, le clic sur un bouton peut déclencher une action comme l'affichage d'une boîte d'information.
